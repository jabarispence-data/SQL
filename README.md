# SQL Portfolio
This repository contains SQL projects that demonstrate my ability to analyze data, write clear and efficient queries, and solve practical business problems. Each project includes the code used to create the dataset, the queries I wrote, and comments explaining my approach.

## What This Portfolio Shows
- Clean, well-structured SQL
- Ability to explore and understand datasets
- Use of filtering, grouping, joins, and aggregations
- Clear reasoning behind each query
- Focus on accuracy and real-world insights

## Goal
To turn raw data into meaningful, actionable information using reliable SQL and straightforward analytical thinking.
